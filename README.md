## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Mohammedyouse/My-resume-.git
```

Navigate to the project:

```bash
cd My-resume-
```

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm run dev
```

Build for production:

```bash
pnpm run build
```
